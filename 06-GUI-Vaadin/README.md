# 06-GUI-Vaadin

## Ziel
Vaadin-Projekt erstellen, sich mit Vaadin Flow vertraut machen und anschließend mehrere Features implementieren, um die Grundlagen von Vaadin zu üben.

## Schritte
### 1. Projekt anlegen

Erstelle ein neues Vaadin-Projekt.
Folge dem offiziellen Getting Started Guide:
https://vaadin.com/docs/latest/getting-started

### 2. Vaadin Flow kennenlernen

Lies die Dokumentation zu Vaadin Flow:
https://vaadin.com/docs/latest/flow


### 3. Grundfunktionalität

Starte das Projekt und überprüfe, ob es läuft.
Implementiere ein Side Navigation Menü, mit dem zwischen verschiedenen Seiten gewechselt werden kann.
Verschiebe den Greet Starter Code auf eine neue Seite, die über das Menü erreichbar ist.


### 4. Dark Mode

Implementiere einen Dark Mode, der das gesamte Layout in eine dunkle Farbpalette umschaltet.
Stelle sicher, dass der Wechsel zwischen Light und Dark Mode einfach möglich ist (z. B. über einen Toggle-Button).


### 5. Chuck Norris Jokes

Füge einen Button hinzu, der bei jedem Klick einen zufälligen Joke von der API https://api.chucknorris.io lädt.
Zeige die Jokes in einer Tabelle an.
Mit jedem weiteren Klick soll ein neuer Joke hinzugefügt werden (nicht ersetzt).
Die Tabelle soll folgende Spalten enthalten:

**Beispiel für die Tabelle:**

| ID                                   | Erstelldatum        | Joke Text                                                        |
|--------------------------------------|--------------------|------------------------------------------------------------------|
| 123e4567e89b12d3a456426655440000     | 2025-11-19T10:00Z  | Chuck Norris counted to infinity. Twice.                         |
| 987f6543c21a43b2b654321098765432     | 2025-11-19T10:05Z  | Chuck Norris can divide by zero.                                 |
| 456a7890b123c456d789e012f3456789     | 2025-11-19T10:10Z  | Chuck Norris doesn’t read books. He stares them down until he gets the information he wants. |

### 6. Kreative Erweiterung – Deine eigene Seite

Werde kreativ und entwickle eine zusätzliche Seite für dein Vaadin-Projekt! Überlege dir einen eigenen Use Case und setze ihn mit den vorhandenen Vaadin-Komponenten um. Kombiniere verschiedene Components
